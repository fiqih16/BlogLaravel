## Installation

- `git clone https://github.com/fiqih16/BlogLaravel.git`
- `cd BlogLaravel`
- `cp .env.example .env`
- `composer updat`
- Edit `.env` 
    - Set your database connection details
- `php artisan migrate`
- `php artisan key:generate`
- `php artisan serve`
